Welcome to Lab 2

This lab focuses on building foundational web programming skills by working with core HTML concepts and page structure. In this activity, you are learning how to create and format web content using proper tags, organize information clearly, and apply basic web development practices that are essential for building functional websites. Overall, the lab helps strengthen your understanding of how web pages are constructed and prepares you for more advanced design and programming tasks later in the course.
